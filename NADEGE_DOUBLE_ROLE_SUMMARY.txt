        FLUX DE TRAVAIL HDJ
═══════════════════════════════════════════════════════════════

1. Patient arrive avec RDV
2. Recherche dans liste du jour
3. Clic "Enregistrer arrivée"
4. Vérification automatique CNAMGS
5. Affichage transparent reste à charge
6. Confirmation enregistrement
7. Génération dossier HDJ
8. Patient en file d'attente

═══════════════════════════════════════════════════════════════
        FLUX DE TRAVAIL URGENCES
═══════════════════════════════════════════════════════════════

URGENCE VITALE (Niveau 1) :
1. Clic bouton rouge "URGENCE VITALE"
2. Saisie rapide nom + motif
3. Création immédiate < 30 secondes
4. Alerte équipe médicale
5. Box réanimation automatique

TRIAGE NORMAL (Niveaux 2-5) :
1. "Nouveau patient"
2. Infos + motif consultation
3. Sélection niveau gravité
4. Constantes vitales si niveau 1-3
5. Placement file attente prioritaire
6. Suivi dashboard Kanban
